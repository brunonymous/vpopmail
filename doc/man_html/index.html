NAME

vpopmail - add on package for qmail to impliment virtual domains under a single user id and group id.

SYNOPSYS

vpopmail

DESCRIPTION

vpopmail is an add on package to qmail for handling virtual email domains on one qmail based email server. It is geared toward ease of use for system administrators as well as security and efficency. With the associated command line programs listed below, system administrators never need touch any of the underlying qmail files and processes. All details are automatically handled.
